About

It's an Angualr application. App using 3-rd party API for getting creatives data.

You can search any creatives by any term. Just type it in search bar.

For run APP just type: `ng serve`

For run TESTS just type: `ng test` or whith params for coverage: `ng test --watch=false --code-coverage` and get visual data form ROOT -> coverage folder -> index.html

For run E2E TESTS just type: `ng e2e`
